# [mare-stare.com](https://mare-stare.com)

Personal website of [@MareStare](https://github.com/MareStare).

## Getting Started

Make sure you have the following tools installed.

1. [`git` CLI](https://git-scm.com/install/) for version control.
2. [`nodejs` toolchain](https://nodejs.org/en/download) for building the site.
3. [VSCode](https://code.visualstudio.com/download) for editing the code.
4. Install VSCode extensions listed in [`.vscode/extensions.json`](.vscode/extensions.json).
   VSCode should prompt you to install them when you open the project via a notification pop-up
   in the bottom right corner of the window.
